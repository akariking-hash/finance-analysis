## 📊 데이터 흐름 예시

### 삼성전자 2024년 재무분석 시나리오

1. **검색**
   ```
   사용자 입력: "삼성"
   → /api/corp-search?q=삼성
   → 삼성전자 (corp_code: 00126380) 선택
   ```

2. **시각화**
   ```
   연도: 2024, 보고서: 사업보고서, 재무제표: 연결
   → /api/financials?corp_code=00126380&year=2024&report_code=11011
   → OpenDart에서 실시간 데이터 조회
   → 매출액: 243.8조 (전년 대비 +1.7%)
   → 영업이익: 58.9조 (전년 대비 +9.9%)
   → 차트와 테이블로 시각화
   ```

3. **AI 분석**
   ```
   주요 계정 데이터 + Gemini 호출
   → "삼성전자의 2024년 재무 현황을 분석하면...
      (비전문가 친화적 분석 텍스트)
   ```

## 🎓 학습 포인트

### 기술 선택 이유
- **Next.js**: Vercel 네이티브 통합, 서버리스 API 자연스러움
- **Recharts**: React 컴포넌트 기반, 반응형 차트
- **fast-xml-parser**: 가벼우면서도 안정적인 XML 파싱
- **Gemini 2.5-Flash**: 최신 모델, 빠른 응답, 한국어 우수

### Vercel 최적화
- 파일 쓰기 제한 → 빌드 시 JSON 생성 (런타임 제외)
- 콜드 스타트 최소화 → 간결한 API 로직
- 환경 변수 활용 → 배포 후 값 변경 가능
